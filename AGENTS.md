# Agents Used
Here are the AI agents I used for this final project:
Claude
Copilot

Here was my AI usage strategy:
I began by giving Claude my idea for a project, to develop a workout tracker. It suggested using python over rust and suggested using FreeSimpleGUI to make a user interface where users can enter and see information.

I then began to code and used Claude to learn how FreeSimpleGUI worked and what functions it included. When I ran code tests, I used Copilot to help me identify what the errors were in order to save me time by scanning google, reddit, and github for error explanations.

I then gave Claude the python file for my tool and asked it to generate a pytest that would run at least 5 tests to verify code integrity. Once it created the pytest, I looked over to make sure it performed the expected functions (ensuring proper syntax, ability to overwrite, ability to delete, etc). It chose all good tests so I implemented the pytest.
